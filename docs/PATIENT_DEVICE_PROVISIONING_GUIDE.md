## Before You Begin

### What is Device Provisioning?

Device provisioning is the process of connecting your PILDHORA device to your account and configuring it for your needs. Once complete, your device will:
- Be linked to your account
- Connect to your WiFi network
- Be ready to manage your medications
- Send you medication reminders

### How Long Does It Take?

The setup wizard takes approximately 10-15 minutes and includes:
- Entering your device ID (2 minutes)
- Verifying device ownership (1 minute)
- Configuring WiFi (3-5 minutes)
- Setting device preferences (3-5 minutes)
- Final confirmation (1 minute)

### Can I Pause and Resume?

Yes! The wizard saves your progress automatically. If you need to stop:
- Your progress is saved
- You can resume later from where you left off
- A "Continue Setup" prompt will appear when you return

### Step 4: WiFi Configuration

Connect your device to your WiFi network.

**What You'll See**:
- WiFi network name (SSID) input
- Password input field
- Security information
- Connection test button

**What to Do**:
1. Enter your WiFi network name (SSID)
2. Enter your WiFi password
3. Tap "Connect"
4. Wait for connection test
5. Tap "Next" when connected

**WiFi Requirements**:
- 2.4GHz or 5GHz network
- WPA2 or WPA3 security
- Minimum 1 Mbps speed
- Stable connection

**Tips**:
- WiFi name is case-sensitive
- Password is case-sensitive
- Check for typos carefully
- Ensure device is in WiFi range

**Connection Test**:
- Takes 30-60 seconds
- Device attempts to connect
- Success: Green checkmark
- Failure: Error message with suggestions

**Troubleshooting WiFi**:

**"Cannot connect to network"**:
1. Verify network name is correct
2. Check password is correct
3. Ensure device is in WiFi range
4. Try moving device closer to router
5. Restart your router if needed

**"Weak signal"**:
1. Move device closer to router
2. Remove physical obstructions
3. Check router is working properly
4. Consider WiFi extender

**"Network not found"**:
1. Verify network name spelling
2. Check router is powered on
3. Ensure network is broadcasting
4. Try 2.4GHz instead of 5GHz

**Security Note**:
- Your WiFi password is encrypted
- Transmitted securely to device
- Never stored in plain text
- Not visible to support staff

### Step 5: Device Preferences

Customize your device settings for alarms and notifications.

**What You'll See**:
- Alarm mode selector
- Volume slider
- LED intensity slider
- LED color picker
- Test alarm button

#### Alarm Mode

Choose how your device alerts you:

**Options**:
- **Sound**: Audible alarm only
- **Vibrate**: Vibration only
- **Both**: Sound and vibration
- **Silent**: Visual LED only

**Recommendations**:
- **Sound**: Best for most users
- **Vibrate**: For quiet environments
- **Both**: For heavy sleepers
- **Silent**: For nighttime medications

#### Volume

Set the alarm volume level.

**Range**: 0-100
- **0**: Silent (LED only)
- **25**: Quiet (soft beep)
- **50**: Medium (normal alarm)
- **75**: Loud (strong alarm)
- **100**: Maximum (very loud)

**Recommendations**:
- Start at 50 and adjust as needed
- Higher for hearing impaired
- Lower for shared spaces
- Test before finalizing

#### LED Intensity

Set the brightness of the LED indicator.

**Range**: 0-100
- **0**: Off
- **25**: Dim (nighttime)
- **50**: Medium (normal)
- **75**: Bright (daytime)
- **100**: Maximum (very bright)

**Recommendations**:
- 50 for normal use
- 25 for bedrooms
- 75 for bright rooms
- 100 for visual impairment

## Troubleshooting

### Setup Issues

#### Cannot Start Setup Wizard

**Problem**: Wizard doesn't open or crashes

**Solutions**:
1. Ensure you're logged in as patient
2. Check internet connection
3. Update app to latest version
4. Restart app and try again
5. Reinstall app if needed

#### Device ID Not Accepted

**Problem**: "Invalid device ID" error

**Solutions**:
1. Verify ID is at least 5 characters
2. Check for typos (case-sensitive)
3. Remove any spaces
4. Use only letters, numbers, hyphens, underscores
5. Try copying and pasting from photo

#### Device Already Claimed

**Problem**: "Device already claimed" error

**Solutions**:
1. Verify you have the correct device ID
2. Check if you already set up this device
3. Contact previous owner to unlink
4. Contact support if you purchased used device

#### WiFi Connection Fails

**Problem**: Cannot connect device to WiFi

**Solutions**:
1. Verify network name (case-sensitive)
2. Check password (case-sensitive)
3. Ensure device is in WiFi range
4. Try 2.4GHz network instead of 5GHz
5. Restart router and try again
6. Check router settings allow new devices

#### Verification Takes Too Long

**Problem**: Stuck on verification step

**Solutions**:
1. Check internet connection
2. Wait up to 2 minutes
3. If still stuck, tap "Cancel"
4. Restart wizard
5. Try again with stable connection

### Post-Setup Issues

#### Device Shows Offline

**Problem**: Device status shows offline after setup

**Solutions**:
1. Check device is powered on
2. Verify WiFi connection
3. Restart device (unplug and replug)
4. Check router is working
5. Reconfigure WiFi if needed

#### Alarms Not Working

**Problem**: Device doesn't alarm at scheduled times

**Solutions**:
1. Verify alarm mode is not "Silent"
2. Check volume is above 0
3. Ensure device time is correct
4. Verify medication schedule is set
5. Test alarm in device settings

#### Cannot Add Medications

**Problem**: Medication wizard doesn't work

**Solutions**:
1. Check internet connection
2. Verify device is online
3. Ensure all required fields are filled
4. Try restarting app
5. Contact support if persists

## Tips for Success

### Best Practices

1. **Keep Device Plugged In**
   - Ensure constant power
   - Avoid battery drain
   - Reliable alarms

2. **Maintain WiFi Connection**
   - Keep device in WiFi range
   - Stable internet connection
   - Regular sync with app

3. **Test Regularly**
   - Test alarms weekly
   - Verify device is online
   - Check battery backup

4. **Update Medications Promptly**
   - Add new medications immediately
   - Update schedules when changed
   - Remove discontinued medications

5. **Review Settings Periodically**
   - Adjust volume as needed
   - Update WiFi if changed
   - Modify preferences seasonally

### Common Questions

**Q: Can I change settings later?**
A: Yes! All settings can be changed anytime in Device Settings.

**Q: What if I move to a new location?**
A: Reconfigure WiFi in Device Settings with your new network.

**Q: Can I have multiple devices?**
A: Currently, one device per patient account. Contact support for special cases.

**Q: What if I lose my device?**
A: Contact support immediately to deactivate and get a replacement.

**Q: How do I update device firmware?**
A: Firmware updates happen automatically when device is online.
