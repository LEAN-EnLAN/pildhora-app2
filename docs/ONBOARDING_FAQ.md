## General Onboarding Questions

### What is onboarding?

Onboarding is the initial setup process when you first use PILDHORA. It includes:
- **For Patients**: Setting up your device (device provisioning)
- **For Caregivers**: Connecting to a patient's device

The process ensures your account is properly configured and ready to use.

### How long does onboarding take?

**Patient Device Provisioning**: 10-15 minutes
- Enter device ID: 2 minutes
- Verify device: 1 minute
- Configure WiFi: 3-5 minutes
- Set preferences: 3-5 minutes
- Complete setup: 1 minute

**Caregiver Connection**: 2-3 minutes
- Enter connection code: 30 seconds
- Verify patient info: 30 seconds
- Establish connection: 1 minute
- Confirmation: 30 seconds

### Can I skip onboarding?

No, onboarding is required to use PILDHORA:
- **Patients** must provision a device to manage medications
- **Caregivers** must connect to at least one patient

However, you can pause and resume the process anytime.

### Can I pause and resume onboarding?

Yes! The onboarding wizard saves your progress automatically:
- Your progress is saved after each step
- You can close the app and return later
- A "Continue Setup" prompt will appear when you return
- Your entered information is preserved

### What if I make a mistake during onboarding?

You can go back and correct mistakes:
- Use the "Back" button to return to previous steps
- Edit information before proceeding
- Cancel and restart if needed
- Contact support for help

### Do I need internet during onboarding?

Yes, internet connection is required:
- To verify device information
- To create device links
- To sync configuration
- To complete setup

Both WiFi and cellular data work, but WiFi is recommended for stability.

## Patient Device Provisioning

### What is device provisioning?

Device provisioning is the process of:
1. Linking your PILDHORA device to your account
2. Connecting the device to your WiFi network
3. Configuring device settings (alarms, LED, etc.)
4. Preparing the device to manage your medications

### When do I provision my device?

You provision your device:
- Immediately after creating a patient account
- When you first log in as a new patient
- The provisioning wizard starts automatically

### What do I need to provision my device?

**Required**:
- Your PILDHORA device (powered on)
- Device ID (found on device label)
- WiFi network name (SSID)
- WiFi password
- Smartphone or tablet with PILDHORA app

**Optional**:
- Device manual for reference
- Pen and paper for notes

### Where do I find my device ID?

The device ID is located:
- On a label on the bottom of the device
- On the back panel
- In the device manual
- On the device packaging

It's usually a sticker with a barcode and alphanumeric code.

### What format is the device ID?

Device ID format:
- Minimum 5 characters
- Maximum 100 characters
- Letters, numbers, hyphens, underscores only
- Case-sensitive
- No spaces

Examples: `DEVICE-001`, `PILDHORA-ABC123`, `DEV_12345`

### Can I use a device that was previously registered?

Yes, but the previous owner must unlink it first:
1. Previous owner opens their PILDHORA app
2. Goes to Device Settings
3. Taps "Unlink Device"
4. Confirms unlinking
5. Wait 5 minutes
6. You can then provision the device

### Can I have multiple devices?

Currently, each patient account can have one device. If you need multiple devices:
- Contact support for special arrangements
- Consider using multiple patient accounts
- Enterprise solutions available for facilities

### What if I don't have a device yet?

You need a PILDHORA device to complete onboarding:
- Purchase a device from pildhora.com
- Contact sales: sales@pildhora.com
- Call: 1-800-PILDHORA
- Check with your healthcare provider

### Can I change device settings later?

Yes! All settings can be changed anytime:
- Go to Settings > Device Settings
- Modify any configuration
- Changes sync to device automatically
- Test settings anytime

## Caregiver Device Connection

### What is device connection?

Device connection is the process of:
1. Linking your caregiver account to a patient's device
2. Gaining access to manage their medications
3. Enabling monitoring and alerts

### When do I connect to a device?

You connect to a device:
- After creating a caregiver account
- When you first log in as a new caregiver
- When adding a new patient to manage

### What is a connection code?

A connection code is:
- A unique 6-8 character code
- Generated by the patient
- Used to link caregiver to patient's device
- Valid for 24 hours
- Single-use only

Example: `ABC123` or `XYZ789AB`

### How do I get a connection code?

The patient generates the code:
1. Patient opens PILDHORA app
2. Goes to Settings > Device Settings
3. Taps "Generate Connection Code"
4. Code is displayed
5. Patient shares code with you

### How long is a connection code valid?

Connection codes expire after 24 hours:
- Use the code within 24 hours
- Complete connection promptly
- Request new code if expired
- Codes cannot be extended

### Can I reuse a connection code?

No, each code is single-use only:
- Once used, code becomes invalid
- Cannot be used again
- Patient must generate new code
- Each caregiver needs their own code

### Can multiple caregivers connect to the same patient?

Yes! Multiple caregivers can connect:
- Each caregiver needs their own code
- Patient generates separate code for each
- All caregivers have equal access
- No limit on number of caregivers

### What access do I have after connecting?

As a connected caregiver, you can:
- ✅ View all medications
- ✅ Add new medications
- ✅ Edit existing medications
- ✅ Delete medications
- ✅ View all events
- ✅ Configure device settings
- ✅ Receive critical alerts

You cannot:
- ❌ Delete patient account
- ❌ Unlink patient's device
- ❌ Change patient's password
- ❌ Generate connection codes

### Can the patient see what I do?

Yes, all actions are logged:
- Medication changes are tracked
- Events show who made changes
- Patient can view event history
- Transparency for patient safety

### Can the patient revoke my access?

Yes, patients can unlink caregivers anytime:
1. Patient goes to Device Settings
2. Views connected caregivers
3. Selects caregiver to remove
4. Taps "Unlink"
5. Confirms action
6. Caregiver loses access immediately

## WiFi and Connectivity

### What WiFi networks are supported?

Supported networks:
- ✅ 2.4GHz networks
- ✅ 5GHz networks
- ✅ WPA2-Personal
- ✅ WPA3-Personal

Not supported:
- ❌ WEP (outdated security)
- ❌ Enterprise networks (WPA2-Enterprise)
- ❌ Captive portals (hotel/airport WiFi)
- ❌ Networks requiring browser login

### Do I need a strong WiFi signal?

Yes, for reliable operation:
- Minimum 1 Mbps speed
- Stable connection
- Device within WiFi range
- Minimal interference

Weak signal can cause:
- Delayed sync
- Missed updates
- Intermittent connectivity
- Configuration issues

### Can I use cellular data instead of WiFi?

No, the device requires WiFi:
- Device doesn't have cellular capability
- Must connect to WiFi network
- Mobile hotspot can work temporarily
- Permanent WiFi recommended

Your phone/tablet can use cellular data to access the app.

### What if my WiFi password changes?

Reconfigure WiFi on the device:
1. Go to Settings > Device Settings
2. Tap "WiFi Configuration"
3. Enter new network information
4. Save configuration
5. Device reconnects automatically

### Can I move my device to a different location?

Yes, but you may need to reconfigure WiFi:
- Same network: No reconfiguration needed
- Different network: Reconfigure WiFi
- Different location: Check WiFi range
- Ensure stable connection

### What if I don't have WiFi at home?

Options:
- Install WiFi router (recommended)
- Use mobile hotspot temporarily
- Check with internet service provider
- Consider cellular-enabled device (future)

WiFi is required for full functionality.

## Device Settings

### What alarm modes are available?

Four alarm modes:
- **Sound**: Audible alarm only
- **Vibrate**: Vibration only
- **Both**: Sound and vibration
- **Silent**: Visual LED only

Choose based on your needs and environment.

### Can I adjust the alarm volume?

Yes, volume is adjustable:
- Range: 0-100
- 0 = Silent (LED only)
- 50 = Medium (recommended)
- 100 = Maximum (very loud)

Test and adjust as needed.

### What is LED intensity?

LED intensity controls the brightness of the alarm light:
- Range: 0-100
- 0 = Off
- 25 = Dim (nighttime)
- 50 = Medium (normal)
- 100 = Maximum (very bright)

Adjust based on lighting conditions.

### Can I choose the LED color?

Yes, you can customize the LED color:
- Choose from preset colors
- Or use custom color picker
- Select any color you prefer
- Test before finalizing

Popular choices: Red (urgent), Green (calming), Blue (neutral)

### How do I test my alarm settings?

Use the test alarm feature:
1. Go to Device Settings
2. Tap "Test Alarm"
3. Device alarms for 5 seconds
4. Verify you can hear/see/feel it
5. Adjust settings if needed
6. Test again until satisfied

### Can I change settings after setup?

Yes, all settings can be changed anytime:
- Go to Settings > Device Settings
- Modify any configuration
- Save changes
- Changes sync automatically
- Test new settings

## Account Management

### Can I change my account type?

No, account type (patient/caregiver) cannot be changed:
- Set during account creation
- Cannot be modified later
- For security and data integrity
- Create new account if needed

### What if I need both patient and caregiver accounts?

Create separate accounts:
- Use different email addresses
- One account for patient role
- Another for caregiver role
- Cannot use same email for both

### Can I delete my account?

Yes, contact support to delete:
- Email: support@pildhora.com
- Include account email
- State reason for deletion
- Response within 48 hours
- Data deleted within 30 days

### What happens to my data if I delete my account?

**Patient Account**:
- Device is unlinked
- Medications are deleted
- Events are deleted
- Caregivers lose access
- Data deleted after 30 days

**Caregiver Account**:
- Device links are removed
- You lose access to patients
- Patient data remains intact
- Other caregivers unaffected

### Can I use the same device with a new account?

Yes, but must unlink first:
1. Unlink device from old account
2. Or factory reset device
3. Create new account
4. Provision device to new account

## Security and Privacy

### Is my data secure?

Yes, PILDHORA uses industry-standard security:
- End-to-end encryption
- Secure cloud storage (Firebase)
- HIPAA-compliant infrastructure
- Regular security audits
- Encrypted WiFi passwords

### Who can see my medication data?

**Patient Data Visible To**:
- You (the patient)
- Connected caregivers
- PILDHORA support (with permission, for troubleshooting only)

**Not Visible To**:
- Other patients
- Unconnected caregivers
- Third parties
- Public

### Are WiFi passwords stored securely?

Yes, WiFi passwords are:
- Encrypted before storage
- Transmitted securely to device
- Never stored in plain text
- Not visible to support staff
- Protected by encryption

### Can I see who accessed my data?

Yes, through the events registry:
- All actions are logged
- Shows who made changes
- Timestamps for all events
- Medication change history
- Device configuration changes

### How are connection codes secured?

Connection codes are secured by:
- Time-limited (24 hours)
- Single-use only
- Randomly generated
- Cannot be guessed
- Logged when used

### What if I lose my phone?

**Immediate Actions**:
1. Change password from another device
2. Log out all sessions
3. Contact support if needed
4. Enable two-factor authentication

**Your Data**:
- Data is stored in cloud
- Not lost with phone
- Access from new device
- Log in with credentials

## Troubleshooting

### What if the wizard doesn't start?

Try these solutions:
1. Log out and log back in
2. Update app to latest version
3. Check internet connection
4. Restart app
5. Reinstall app if needed

### What if my device ID isn't accepted?

Check these common issues:
- Minimum 5 characters required
- No spaces allowed
- Only letters, numbers, hyphens, underscores
- Case-sensitive
- Verify from device label

### What if WiFi connection fails?

Troubleshoot WiFi issues:
1. Verify network name (case-sensitive)
2. Check password (case-sensitive)
3. Ensure device in WiFi range
4. Try 2.4GHz instead of 5GHz
5. Restart router if needed

### What if device shows offline after setup?

Check these items:
1. Device is powered on
2. WiFi connection is active
3. Router is working
4. Device is in WiFi range
5. Restart device

### What if connection code doesn't work?

Verify these details:
1. Code is correct (check for typos)
2. Code hasn't expired (< 24 hours)
3. Code hasn't been used
4. Format is correct (6-8 characters)
5. Request new code if needed

### What if I can't add medications after setup?

Check these requirements:
1. Device is online
2. Internet connection active
3. All required fields filled
4. Valid dosage and schedule
5. Restart app if needed
