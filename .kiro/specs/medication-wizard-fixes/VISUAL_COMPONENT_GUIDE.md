### Common Spacing Patterns

**Card Padding:**
- Small screens: 12px (md)
- Medium screens: 16px (lg)
- Tablets: 24px (xl)

**Section Margins:**
- Between sections: 24px (xl)
- Between elements: 12px (md)
- Between related items: 8px (sm)

**Grid Gaps:**
- Emoji grid: 6-12px (sm-md)
- Chip grid: 8-12px (sm-md)
- Type grid: 12px (md)

### Timing

```
Fast:   150ms  (Hover states, quick feedback)
Normal: 300ms  (Transitions, state changes)
Slow:   500ms  (Page transitions, complex animations)
```

### Easing

```
ease-in:     Start slow, end fast
ease-out:    Start fast, end slow (preferred for UI)
ease-in-out: Smooth start and end
linear:      Constant speed (for continuous animations)
```
