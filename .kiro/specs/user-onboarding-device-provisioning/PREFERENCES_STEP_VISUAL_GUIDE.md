## Responsive Design

- Alarm mode grid: 2x2 layout on all screen sizes
- Sliders: Full width with padding
- Buttons: Full width, stacked vertically
- Color picker: Modal overlay, centered

## Color Palette

### Alarm Mode Cards
- Default: `colors.surface` with `colors.gray[200]` border
- Selected: `#EFF6FF` (primary[50]) with `colors.primary[500]` border

### LED Preview
- Background: `colors.gray[900]`
- LED indicator: User-selected color with intensity-based opacity

### Buttons
- Primary: `colors.primary[500]`
- Secondary: `colors.gray[200]`
- Disabled: Reduced opacity

### Feedback
- Success: `#F0FDF4` (success[50]) background
- Error: `#FEF2F2` (error[50]) background
- Info: `#EFF6FF` (primary[50]) background

## Touch Targets

All interactive elements meet minimum touch target size:
- Buttons: 44x44 minimum
- Alarm mode cards: 56x56 icon area
- Color swatches: 56x56
- Sliders: 40px height

## Performance Considerations

- Color picker modal: Lazy loaded
- Slider updates: Debounced for smooth performance
- Form data updates: Batched with useEffect
- LED preview: CSS-based, no re-renders needed
